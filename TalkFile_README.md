# 방식 A — 결혼상태 × (나이·성별·유전위험) 교호작용 검정

알츠하이머(AD) 진단 데이터에서 **"결혼해체(사별·이혼)의 AD 위험 효과가 나이·성별·ApoE4에 따라 달라지는가?"** 를 로지스틱 회귀의 **교호작용(interaction)** 으로 검증한다.

> 관련 노트북: [`alzheimer-analysis-kaggle.ipynb`](alzheimer-analysis-kaggle.ipynb) 의 **섹션 10 (셀 A1~A5)**

---

## 1. 아이디어

"결혼상태의 효과가 나이/성별/유전위험에 따라 달라지는가"를 본다. 요소를 **곱해서(interaction)** 모형에 넣는다.

```
logit P(AD) = β0 + β1·결혼해체 + β2·Age + β3·(결혼해체 × Age)
                 + 성별 + 교육 + ApoE4 + ...
```

- **β3(교호항)** 가 유의하면 → *"사별·이혼이 고령일수록 AD 위험을 더 키운다"* 는 **결합효과**가 입증된다.
- 성별 교호(결혼해체 × 성별), 유전 교호(결혼해체 × ApoE4)도 같은 방식으로 함께 검정한다.
- **장점:** '독거' 프록시 없이 나이+결혼상태 결합효과를 정면으로 검증하므로 **순환 문제가 없다.** 통계적으로 가장 정석적인 접근.

---

## 2. 변수 정의

| 변수 | 정의 | 비고 |
|------|------|------|
| **결과 `AD`** | `Diagnosis == 'AD'` → 1, 그 외 → 0 | 이진 결과 |
| **`Dissolved` (결혼해체)** | 사별(Widowed)·이혼(Divorced) = 1, 기혼(Married) = 0 [기준] | `Never married`/`Unknown`은 주분석 제외 |
| **`Age_c`** | 나이 − 평균 (중심화) | 주효과를 평균 지점에서 해석 |
| **`Female`** | 여성 = 1, 남성 = 0 | |
| **`Educ_c`** | 교육연수 − 평균 (중심화) | 교란 보정 |
| **`ApoE4_c`** | ApoE4 대립유전자 수(0/1/2) − 평균 | 유전위험 |

**중심화(centering)**: 연속변수(Age·Educ·ApoE4)를 평균 기준으로 빼줌으로써, 교호모형의 *주효과 계수*가 "평균값 지점에서의 효과"로 해석되도록 한다.

---

## 3. 분석 절차 (노트북 셀 구성)

| 셀 | 내용 |
|----|------|
| **A1** | 분석용 데이터 구성 (결과·결혼해체·공변량 정의, 중심화) |
| **A2** | 주효과 모형 vs 교호작용 모형 로지스틱 회귀 → OR·95% CI·p값 표 |
| **A3** | β3 교호항 개별 유의성 + 3개 교호항 결합 우도비(LR) 검정 |
| **A4** | 나이에 따른 예측 AD 확률 곡선 (기혼 vs 결혼해체, 95% CI) |
| **A5** | 해석 가이드 & 주의사항 |

두 모형의 우도비 검정(LR test)으로 "교호항을 넣는 것이 모형을 유의하게 개선하는가"를 판정한다.

---

## 4. 결과

**분석 표본:** N = 1,666 (사별 206 + 이혼 150 vs 기혼 1,310), AD 334명

### 교호항 (β3) 검정

| 교호항 | OR | p-value | 판정 |
|--------|----|---------|------|
| **결혼해체 × 나이** | **1.064** | **0.025** | ★ 유의 |
| 결혼해체 × 여성 | 0.752 | 0.460 | n.s. |
| 결혼해체 × ApoE4 | 0.879 | 0.663 | n.s. |

- 결합 우도비 검정: LR χ² = 7.76, df = 3, **p = 0.051** (경계선)

### 핵심 결론

- ✅ **결혼해체 × 나이 교호항이 유의(OR=1.064, p=0.025).** 즉 **"고령일수록 사별·이혼이 AD 위험을 더 키운다"** 는 결합효과를 데이터가 지지한다. A4 그래프에서 두 곡선이 고령으로 갈수록 벌어지는 것으로 시각적으로 확인된다.
- 결혼해체 효과는 **나이에는 조절되지만, 성별·유전위험(ApoE4)에는 뚜렷한 조절이 없다.**

---

## 5. 해석 시 주의사항

- **횡단(cross-sectional) 진단 자료**이므로 인과가 아닌 **연관**으로 해석해야 한다.
- 주효과에서 결혼해체 OR < 1로 나오는 것은 교란(기혼자가 평균적으로 더 고령·다른 구성 등) 가능성이 크다. 반드시 **교호작용 예측곡선(A4)** 과 함께 읽는다.
- 결혼해체 주효과 OR은 *평균 나이 지점*에서의 값이며, 교호가 있으면 나이에 따라 방향·크기가 달라진다.

### 권장 민감도 분석
1. 결과를 AD 대신 **(LMCI + AD)** 로 확장
2. ApoE4를 **보인자 여부(≥1)** 이진 변수로 변경
3. **'Never married' 포함 3범주** 결혼상태로 재확인
